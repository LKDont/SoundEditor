# SoundEditor
基于FFmpeg的声音编辑器。



## FFmpeg
编译脚本来源于[ijkplayer](https://github.com/bilibili/ijkplayer)，添加了https协议、mp3lame



## 实现功能
1. PCM音频的格式转换
2. Mp3、AAC解码 
3. Mp3、AAC编码
4. Mp3、AAC播放器
